# pymakr

This directory contains MicroPython projects for ESP32 boards, intended for use with the Pymakr plugin and VS Code.

## Structure
- `esp32_blink_test/` - Example for blinking an LED on ESP32
- `esp32_project/` - Main MicroPython project for ESP32
- `esp32v2.ps1` - PowerShell script for ESP32 setup

## Requirements
- MicroPython firmware on ESP32
- Pymakr VS Code extension

## Usage
Open the desired project folder in VS Code with Pymakr installed. Upload and run scripts on your ESP32 board.
